# 训练不了模型就训练自己

提出better prompt

1. 使用模型思考，在prompt中加入let's think step by step
2. 使用情绪绑架，在prompt中加入，this is important for my career等等
3. 提供前提
    - 比如，让他作为一个什么样的角色
    - 限定范围
4. 提供范例（in-text learning）
    - 给出一段你自己制定的规则，然后带着这个规则丢给模型
    - 带着教科书和字典，丢给模型

重要的观念就是：不管怎么样修改prompt或者是改进prompt都不会有训练模型的动作
